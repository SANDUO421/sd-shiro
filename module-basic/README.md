## 设计理念
 ```
设计理念：
	数据库设计：
		分布式（主键ID的生成）
		三大范式
	代码设计：
		23种设计模式
	系统架构层面
		三层模式：(1.0)
			数据处理->(基础支撑组件+业务间组件)->应用
		五层模式:(2.0)
			数据处理->数据中台->(基础支撑组件+业务间组件)->业务中台->应用中台
考虑：
	限流、并发、缓存
```
